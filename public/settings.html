How to get an API key

Claude (Anthropic)

  1. Go to console.anthropic.com
  2. Create an account
  3. Go to API Keys
  4. Create a new key
  5. Copy and paste it above

OpenAI

  1. Go to platform.openai.com
  2. Create an account
  3. Go to API Keys
  4. Create a new secret key
  5. Copy and paste it above

Gemini (Google)

  1. Go to aistudio.google.com
  2. Sign in with Google
  3. Click "Get API Key"
  4. Create a key
  5. Copy and paste it above
